OKLCH 값 설명

  • L (Lightness): 0 (검은색) ~ 1 (흰색)
  • C (Chroma): 0 (무채색) ~ 0.4 (고채도)
  • H (Hue): 0 ~ 360 (색상각)

예: oklch(.145 0 0) = 밝기 14.5%, 채도 0 (회색), 색상 0
